AI 기반 취업 준비생 케어 플랫폼  

“당신의 컨디션에 맞춘 개인화 커리어 성장 파트너”

<br><br><br>

## 📌 프로젝트 소개

CarYou는 취업 준비 과정에서 겪는 번아웃, 방향성 부족, 정보 과잉 문제를 해결하기 위해 만든  
AI 기반 커리어 관리 플랫폼입니다.

단순한 채용 정보 제공을 넘어,

- 직무 기반 AI 학습 콘텐츠  
- 감정 상태 기반 루틴 추천  
- 개인화 커리어 성장 시스템  

을 통해 지속 가능한 취업 준비 경험을 제공합니다.

## ✨ 핵심 기능

### 1️⃣ AI 직무 맞춤 콘텐츠 (Upstage AI)
- 일일 키워드 추천  
- 3분 산업 리포트 생성  
- 직무 기반 학습 콘텐츠 제공  

<br>

### 2️⃣ 사용자 감정 기반 커리어 루틴 추천
- 스트레스 상태에 따라 학습 난이도 자동 조절  
- 프롬프트 엔지니어링 기반 맞춤 루틴 생성  

| 스트레스 레벨 | 추천 활동 |
|------|------|
| High | 명상, 산책, 짧은 학습 |
| Middle | 코드 리뷰, 튜토리얼 |
| Low | 프로젝트, 심화 학습 |

> 단순 추천이 아닌 컨디션 기반 적응형 시스템

<br>

### 3️⃣ 채용 공고 추천
- 관심 직무 기반 필터링  
- 최신 공고 제공  
- 마감일 및 상세 정보 관리  

> 공공 API 문제를 해결하기 위해 DB 기반 필터링 시스템으로 개선

<br>

### 4️⃣ 활동 기록 & 캘린더
- 학습 기록 아카이빙  
- 일별 활동 추적  
- 성장 흐름 시각화  

<br><br><br>

## 🏆 차별화 포인트

### 🧠 3단계 AI 개인화 시스템
- 직무 설정 → 콘텐츠 생성 → 감정 기반 난이도 조절  

<br>

### 💚 감정 케어 기반 학습
- 컨디션에 맞춘 AI 학습 루틴 제공  
- 번아웃 방지  
- 지속 가능한 성장
